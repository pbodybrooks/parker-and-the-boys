Want to get away, but don't know where to go?

No problem, Parker and the Boys have got you covered. Simply enter the weather criteria and distance you're willing to drive to get the weather for cities within the united states. From there, pick a city from the generated list and enjoy your weekend!

How does it work?

This project was built using ~400 lines of JavaScript to tackle the functionality, and the Pico framework to aid in structure and styling. Before use, the webpage will automatically run a parent function to prepare the application. First, the user's current location is gathered. Next, the function will get weather information for a predefined bank of cities that are saved to local storage to prevent having to fill the weather bank each time the user visits the page. The webpage uses two APIs; one to return the user's current position, and another to return each cities' position and corresponding weather. From these two sets of coordinates, the Haversine method, used to calculate the distance between two points on a sphere, is used to approximate the distance between the user and the returned cities that match their desired weather and range criteria.
